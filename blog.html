What is the Document Object Model (DOM)?

The Document Object Model (DOM) is like a blueprint for a webpage. It organizes all the elements (like text, images, buttons) into a structured tree, where each part of the page is represented as an object. This lets you use JavaScript to change, add, or remove parts of the page dynamically. For example, when you click a button to load new content without reloading the page, the DOM is what allows that to happen.

How do you select an element from the DOM?

To select an element from the DOM using JavaScript, you can use various methods:


  • 1. document.getElementById("id"): Selects an element by its id attribute.
  • 2. document.getElementsByClassName("class"): Selects elements by their class attribute (returns a live HTMLCollection).
  • 3. document.getElementsByTagName("tag"): Selects elements by their tag name (returns an HTMLCollection).
  • 4. document.querySelector("selector"): Selects the first element that matches a CSS selector (id, class, tag, etc.).
  • 5. document.querySelectorAll("selector"): Selects all elements that match a CSS selector (returns a static NodeList).
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you attach a single event listener to a parent element to manage events for multiple child elements, even if they are added dynamically. Instead of adding individual listeners to each child, the parent listens for events that bubble up from the children (thanks to event propagation).


This is useful because:


  • Efficiency: Instead of attaching multiple event listeners, you use a single one on the parent element. This is particularly beneficial when handling large numbers of elements.
  • Dynamic content: It allows handling events on elements added to the DOM after the initial page load, which wouldn't have event listeners if added dynamically.
How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using JavaScript through the DOM.


Manipulating Attributes:


  • 1. element.setAttribute("attribute", "value"): Adds or changes the specified attribute.
  • 2. element.getAttribute("attribute"): Retrieves the value of the specified attribute.
  • 3. element.removeAttribute("attribute"): Removes the specified attribute.

Manipulating Styles:

You can manipulate an element's inline styles using the style property


  • 1. element.classList.add("class-name"): Adds a class.
  • 2. element.classList.remove("class-name"): Removes a class.
  • 3. element.classList.toggle("class-name"): Toggles a class on and off
